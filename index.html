    "I want to learn how to code/develop websites! Where can I start?"

    • Codecademy has tutorials over many, many languages and APIs
    • Python is simple to learn yet immensely powerful and is used world-wide by scientists and STEM industries

    • Stack Overflow will have the answer to nearly any coding issue
    • Google is always your friend

    "I saw that button at the top. What is GitHub?"

    GitHub is a place to store and edit code for various projects. The company provides version control to keep track of changes automatically and allow more than one person to work on a project at the same time. Google Docs is for writing papers; GitHub is for writing software.
